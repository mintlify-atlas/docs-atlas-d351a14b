### Factorial with non-integers

For non-integer inputs, `x!` uses the Lanczos approximation of the gamma function: `О“(x + 1)`. Integer factorials up to 2999 are computed exactly using `BigInteger` arithmetic. At 3000 or above, the result is reported as infinity.

## Special syntax

### Implicit multiplication

OpenCalc automatically inserts a `Г—` operator when the expression would otherwise be ambiguous:


### Percentage

The `%` operator behaves differently depending on context:

| Expression | Interpretation | Result |
|---|---|---|
| `50%` | Standalone: divides by 100 | `0.5` |
| `100 + 10%` | After `+` or `-`: relative to base | `110` |
| `200 Г— 10%` | After `Г—` or `Г·`: plain division by 100 | `20` |

### Scientific notation input

Use `E` to enter numbers in scientific notation:

```text
1.5E3  в†’  1.5 Г— 10^3  в†’  1500
```

### Auto-closing parentheses

If an expression has unclosed parentheses (e.g., `sin(30`), OpenCalc automatically appends the missing `)` before evaluation.
