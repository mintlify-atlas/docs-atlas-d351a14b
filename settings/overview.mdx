OpenCalc exposes all of its settings through a single in-app screen. Changes take effect immediately — no restart required.

## Opening settings

<Steps>
  <Step title="Open the overflow menu">
    Tap the three-dot menu icon (**⋮**) in the top-right corner of the calculator.
  </Step>
  <Step title="Tap Settings">
    Select **Settings** from the dropdown menu.
  </Step>
</Steps>

## Settings categories

<CardGroup cols={2}>
  <Card title="Display" icon="display" href="/settings/display">
    Control how numbers and the interface look — theme, number precision, scientific notation, and scientific mode panel visibility.
  </Card>
  <Card title="Behavior" icon="sliders" href="/settings/behavior">
    Configure how the calculator acts — haptic feedback, history limits, angle units, button layout, and lock-screen access.
  </Card>
</CardGroup>

## All settings at a glance

| Setting | Category | Default |
|---|---|---|
| Theme | Display | System |
| Numbering system | Display | International |
| Number precision | Display | 10 |
| Scientific notation | Display | Off |
| Scientific mode | Display | Shown, collapsed |
| Vibration | Behavior | On |
| History size | Behavior | 50 |
| Delete history on swipe | Behavior | On |
| Auto-save calculation | Behavior | On |
| Prevent phone from sleeping | Behavior | Off |
| Long click to copy value | Behavior | On |
| Use radians by default | Behavior | Off (degrees) |
| Add modulo button | Behavior | On |
| Split parenthesis button | Behavior | Off |
| Move back button left | Behavior | Off |
| Show on lock screen | Behavior | On |

<Note>
  The **App language** option appears only on Android 13 and later. It opens the system-level per-app language selector.
</Note>
